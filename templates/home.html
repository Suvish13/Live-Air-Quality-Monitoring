LIVE AIR QUALITY MONITORING APPLICATION

This application was built to visualize live air quality parameterssuch as PM2.5, PM10, SO2, CO, CO2, O3, NO2, Relative Humidity, and Temperature using free and open data from the OpenAQ API.The goal is to create an interactive and user-friendly application for understanding the current air quality across Southern India in real time through a map.Users can click directly on map markers to view the real-time pollutant values, and they also have the option to visualize the data in chart form for better insights.

I initially began by exploring how live air quality data could be visualized, since my earlier research work focused mainly on mapping air pollutants using Sentinel-5P satellite imagery. However, I wanted to move beyond satellite-based analysis and experiment with real-time monitoring. During this exploration, I found the OpenAQ API, a free source that provides live air quality data. I studied their documentation and sample code through Google Colab to understand the JSON data structure they provide. After understanding the data structure, I used Flask to build a web application for visualization. To make it more interactive, I integrated Leaflet.js for mapping the monitoring stations and displaying pollutant values in real time.
